# Crust Restful API

### Build
```shell
yarn
```

### Start
The default startup will start with 56666 API port and connect to the local Crust chain:
```shell
yarn start
```
Change the setting like this, if you want to connect to other chain:
```shell
yarn start 56666 ws://192.168.50.6:9944/
```
